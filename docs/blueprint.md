# **App Name**: TriageAssist

## Core Features:

- Message Categorization: Automatically categorize support messages based on keywords (Bug, Billing, Feature Request, General).
- Priority Assignment: Assign priority levels (High, Medium, Low) to messages based on predefined rules. Visual cues for high priority.
- Summary Statistics Display: Display the count of messages by category and priority in a summary section at the top.
- Filterable Message List: Display messages in a filterable list/table with message snippet, category, priority, and status. Filter by category and priority.
- Message Resolution: Option to mark messages as 'Resolved', greyed out or hidden via toggle.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to give a calm, reliable feel.
- Background color: Light grey (#F0F4F8), near white, to keep the focus on content and data.
- Accent color: Pale violet (#BDB2FF), approximately 30 degrees counter-clockwise on the color wheel, used for highlights.
- Body and headline font: 'Inter' (sans-serif) for a clean and readable layout.
- Simple, clean layout with summary at the top and message list below. High-priority messages visually emphasized.
- Simple icons to represent message categories and priority levels. For instance, exclamation marks for priority and familiar symbols for category (a bug symbol for a bug).
